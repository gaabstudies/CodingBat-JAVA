# CodingBat code practice

-CodingBat é um site gratuito de problemas de codificação ao vivo para desenvolver habilidades de codificação em Java e Python 
-Passar por muitos problemas de prática é uma ótima maneira de solidificar sua compreensão de como o código deve funcionar. 
-Os problemas do CodingBat funcionam muito bem como lição de casa, ou para a prática de auto-estudo. 
-s problemas do CodingBat são projetados para ter baixa sobrecarga: declarações curtas de problemas (como um exame), nada para instalar, feedback imediato no navegador e muitos deles para desenvolver essas habilidades.
